Open IncrementalCompiler.sln
Select Release configuration
Click Build > Rebuild Solution
delete /bin
run ./build.cmd export

this will build the project and move all needed files to bin/UnityCompiler folder.

to move these files to sample project, navigate to samples/Basic and run InstalCompilerNotBundled.bat

to debug project from samples/Basic in Rider,
open IncrementalCompiler solution, create new run configuration with these program arguments:
-dev C:/work/unity/Unity3D.IncrementalCompiler/samples/Basic <UnityTempFile>
<UnityTempFile> is generated by unity when compiler ir ran to samples/Basic/Temp directory
